MathJax에서 툴팁

이 페이지는 요약, 확장 기능 그리고 2011년 4월에 MathJax Users Group 에서 ‘Labeling equation terms’ 를 주제로 한 실용 예제이다.
Davide P. Cervone 에 의해 기고된 해법들을 포함한다.

툴팁 위치에 영향을 미치는 알려진 버그가 있지만 곧 고쳐질 것이다.

툴팁의 네가지 버전이 있다:

  • 기본 툴팁:
    첫 번째 인수는 클릭할 수 있는 부분이다;
    두 번째 인수는 기본 수식 내용의 툴팁이다.
    \tooltip{#1}{#2}
    보기:
    \sum_{n=0}^\infty {\tooltip{x}{\text{converges for }|x|\lt 1 }}^n = \frac{1}{1-x}
    결과 (아래 표현식 $x^n$ 의 ‘$x$’ 위에 나타난다)

    $ \displaystyle \sum_{n=0}^\infty {\tooltip{x}{\text{converges for }|x|\lt 1}}^n = \frac{1}{1-x} $
  • 컬러가 있는 기본 툴팁:
    첫 번째 인수는 선택된 컬러로 식별된 클릭할 수 있는 부분이다;
    두 번째 인수는 기본 수식 내용의 툴팁이다
    \tooltipc{#1}{#2}
    보기:
    \sum_{n=0}^\infty {\tooltipc{x}{\text{converges for }|x|\lt 1 }}^n = \frac{1}{1-x}
    결과 (아래의 녹색 ‘$x$’ 위에 나타난다)

    $ \displaystyle \sum_{n=0}^\infty {\tooltipc{x}{\text{converges for }|x|\lt 1}}^n = \frac{1}{1-x} $
  • 텍스트 툴팁:
    첫 번째 인수는 클릭할 수 있는 부분이다;
    두 번째 인수는 기본 텍스트 내용의 툴팁이다.
    \texttip{#1}{#2}
    두 번째 인수는 수식이 적절한 구분자를 포함하는 한 수식을 포함할 것이다.

    보기:
    \texttip{a}{the first term is an \(a\)} \texttip{+}{a binary operator} \texttip{b}{the second term}
    결과 (아래 표현식의 각 부분에 나타난다)

    $ \texttip{a}{the first term is an \(a\)} \texttip{+}{a binary operator} \texttip{b}{the second term} $


    보기:
    \texttip{a}{the first term is an \(a\)} \mathbin{\texttip{+}{a binary operator}} \texttip{b}{the second term}
    결과 (아래 표현식의 각 부분에 나타난다)

    $ \texttip{a}{the first term is an \(a\)} \mathbin{\texttip{+}{a binary operator}} \texttip{b}{the second term} $

    두 번째 보기에서,   \mathbin   은 texttip 된 플러스 기호가 간격이 늘어난 이항 연산자가 되게 한다.
  • 컬러가 있는 텍스트 툴팁:
    첫 번째 인수는 선택된 컬러로 식별된 클릭할 수 있는 부분이다;
    두 번째 인수는 기본 텍스트 내용의 툴팁이다.
    \texttipc{#1}{#2}
    두 번째 인수는 수식이 적절한 구분자를 포함하는 한 수식을 포함할 것이다.

    보기:
    \frac{\texttipc{a+b}{numerator}}{\texttipc{2}{denominator}}
    결과 (아래 표현식의 각 녹색 부분에 나타난다)

    $ \displaystyle \frac{\texttipc{a+b}{numerator}}{\texttipc{2}{denominator}} $

비록 이 기능이 MathJax에 만들어지지 않았지만, 이 페이지의 방법을 따르면
(그리고 이 페이지의 예제에 있는 소스코드를 사용하면) 쉽게 이용할 수 있다.

  • 1 단계:
    문서의 구성 영역에 다음을 포함한다.
    (동작을 보려면 이 페이지의 소스코드를 보시오.) 
    MathJax.Hub.Register.StartupHook("TeX Jax Ready",function () {
  var TEX = MathJax.InputJax.TeX,
    MML = MathJax.ElementJax.mml;
  TEX.Definitions.macros.tooltip = "myToolTip";
  TEX.Parse.Augment({
    myToolTip: function (name) {
      var arg = this.ParseArg(name), tip = this.ParseArg(name);
      this.Push(MML.maction(arg,tip).With({actiontype: MML.ACTIONTYPE.TOOLTIP}));
    }
  });
});
  • 2 단계:
    MathJax.Hub.Config 안에 다음을 포함한다.
    (동작을 보려면 이 페이지의 소스코드를 보시오.) 
    TeX: {
  Macros: { 
    tooltipc: ["\\tooltip{\\color{green}{#1}}{#2}",2],
    texttip: ["\\tooltip{#1}{\\text{#2}}",2],
    texttipc: ["\\tooltip{\\color{green}{#1}}{\\text{#2}}",2]
  }
}
    구현하기 원하는 매크로를 위해서 라인들을 포함시키는 것이 필요하다.
    컬러를 바꾸기 원하면 ‘green’ 을 바꾸기만 하면 된다.
